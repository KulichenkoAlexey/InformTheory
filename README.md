Этот проект предназначен для изучения жадных алгоритмов, в частности - алгоритма нахождения минимального оставного дерева.
---
При запуске из main.py, с заданным параметром числа вершин, создает окно с графическим представлением случайного графа с таким числом вершин, который является минимальным оставным графом для случайно сгенерированных при запуске вершинах.
